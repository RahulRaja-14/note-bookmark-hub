## Key Components

### Frontend (React)
- `Auth.tsx` - Authentication UI with multi-step form
- `useAuth.ts` - Authentication hook with methods
- `ProtectedRoute.tsx` - Route protection component

### Backend (Supabase)
- `auth.users` - User authentication table
- `public.profiles` - User profile data
- `public.notes` - User notes
- `public.bookmarks` - User bookmarks

### Security
- Email OTP verification
- Row Level Security (RLS)
- Automatic session management
- Secure password storage
- Token auto-refresh
